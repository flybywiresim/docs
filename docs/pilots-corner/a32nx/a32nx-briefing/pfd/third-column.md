## Description

These annunciations tell the pilots which lateral mode the autopilot and the flight director are engaged in and which modes are armed.

---

### RWY

Displayed in green when runway mode is engaged. This is only displayed if there is an active localizer signal. This mode provides lateral guidance during the takeoff roll up to 30 feet (radio altitude).

### RWY TRK

Displayed in green when runway mode is engaged once airborne or above 30 ft RA

### HDG

Displayed in green when heading mode is engaged. Heading mode is when the aircraft follows the FCU selected heading (selected mode).

### TRACK

Displayed in green when track mode is engaged. Track mode is when the aircraft follows the FCU selected track (selected mode).

### NAV

- Displayed in green when NAV mode is engaged to guide the aircraft along the lateral flight plan (managed mode).
- Displayed in blue when NAV mode is armed.

### LOC\*

Displayed in green when localizer capture mode is engaged.

### LOC

- Displayed in green when localizer track mode is engaged.
- Displayed in blue when LOC mode is armed.

### APP NAV

- Displayed in green when NAV mode is engaged during a non ILS approach.
- Displayed in blue when NAV mode is armed during a non ILS approach.

### GA TRK

Displayed in green when go around track mode is engaged.
